# ДЕТАЛИ | Студия премиум-детейлинга

Современный, быстрый и отзывчивый одностраничный веб-сайт (landing page) для студии автомобильного детейлинга представительского класса.

## 🚀 Технологический стек
* **Фреймворк:** React (v19)
* **Язык:** TypeScript
* **Сборщик:** Vite
* **Стилизация:** TailwindCSS (v4)
* **Анимации:** Framer Motion & Lucide React

## ✨ Особенности проекта
* **Интерактивный слайдер «До / После»:** наглядное визуальное сравнение состояния лакокрасочного покрытия до полировки и после нанесения защитных составов.
* **Интерактивный калькулятор:** выбор класса автомобиля и пакета услуг с динамическим расчетом стоимости и выводом технологической карты.
* **Кинематографичный Hero-блок:** фоновое 60fps-видео с Porsche, которое проигрывается вперед, затем в реверсе и останавливается для оптимизации производительности устройства.
* **100/100 в AISLOP:** код полностью очищен от предупреждений, ИИ-мусора и отладочных логов.
* **Адаптивность:** идеальный интерфейс для мобильных устройств, планшетов и ПК.

## 🛠️ Запуск проекта локально
1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите сервер разработки:
   ```bash
   npm run dev
   ```
3. Сборка для продакшна:
   ```bash
   npm run build
   ```
