# ДЕТАЛИ

Сайт студии премиального детейлинга «Детали», разработанный на React, TypeScript и TailwindCSS с быстрой сборкой на Vite. 

Благодаря оптимизированной буферизации медиафайлов и отключению цикличного фонового воспроизведения видео после первого прокрута назад, сайт работает максимально плавно и не перегружает процессор даже на мобильных устройствах.
